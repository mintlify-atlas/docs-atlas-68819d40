## What You'll Build

Over four progressive exercises, you'll go from a simple agent with a weather tool to a fully durable, multi-language routing system backed by Temporal:

<Steps>
  <Step title="Agent Hello World">
    Create your first OpenAI agent with a real weather tool that calls the National Weather Service API. Learn how LLMs decide when to invoke tools.
  </Step>
  <Step title="Temporal Hello World">
    Write your first Temporal workflow and activity. Watch execution in the Temporal UI and experience automatic retries.
  </Step>
  <Step title="Durable Agent">
    Combine the OpenAI Agents SDK with Temporal to wrap LLM calls in activities — getting retries, crash recovery, and full observability for free.
  </Step>
  <Step title="Agent Routing">
    Build a multi-agent triage system that routes queries to French, Spanish, or English specialist agents using the handoff pattern, structured as a production-ready Temporal application.
  </Step>
</Steps>

## Why Temporal for AI Agents?

<CardGroup cols={2}>
  <Card title="Automatic Retries" icon="arrows-rotate">
    LLM API calls wrapped in Temporal activities retry automatically on failure with configurable backoff policies
  </Card>
  <Card title="Crash Recovery" icon="shield-check">
    Workflows persist their state — if your worker crashes mid-execution, Temporal resumes exactly where it left off
  </Card>
  <Card title="Full Observability" icon="chart-line">
    Every activity, handoff, and LLM call appears in the Temporal Web UI with complete execution history
  </Card>
  <Card title="Zero Agent Changes" icon="wand-magic-sparkles">
    Your OpenAI agent code stays identical — Temporal wraps it with production superpowers via a plugin
  </Card>
</CardGroup>

## Workshop Structure

This is a **90-minute workshop**: 30 minutes of instruction followed by four 15-minute hands-on exercises.

**During the workshop** — work through the complete, running implementations in `solutions/`. These are fully working examples you'll execute, observe, and learn from.

**After the workshop** — practice building everything from scratch using the starter code in `exercises/` as optional homework.

<Note>
All exercises run in GitHub Codespaces. You need a GitHub account and an OpenAI API key (~$0.50–$1.00 of credits for the full workshop).
</Note>
